magpie

An open source MATLAB library for working with thin plates with generalized elastic boundary conditions. The framework is designed around two key use cases

  1. Exploration of plates for understanding how material properties affect mode shapes and modal frequencies.
  2. Derivation of material properties from experimental data.

Functions

  • magpie: modal analysis of 2D plates with generalised elastic boundary conditions
  • bhmat: generate the 2D biharmonic operator with elastic boundary constants.
  • <./youngcalc_help.htmlyoungcalc: estimate Young's modulus from measured modal frequencies >
  • biharmdiag: biharmonic coefficients of a diagonal band by sublock
  • fidimat: generate finite difference spatial sparcity matrices
  • bhcoefs: biharmonic coefficients of a block order
